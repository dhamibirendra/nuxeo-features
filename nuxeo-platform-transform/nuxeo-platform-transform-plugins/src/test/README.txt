Before running the tests you will need to start an openoffice server on
port 8100 on the host running the tests as below:

$ /opt/ooo-dev2.0/program/soffice -headless -accept="socket,port=8100;urp;"

It's been tested against OOo 2.x.

With Windows:

Go to your OpenOffice folder (for example : D:\Program Files\OpenOffice.org 2.0).
Write the following commandline: soffice -headless -accept="socket,port=8100;urp"
